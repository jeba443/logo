# Heart Break Animation - React Project

A fully animated emotional heart-break card made using React, CSS, and JavaScript logic.

## Run the project

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## What is inside

- Animated glowing heart
- Click to break heart into two parts
- Floating pieces and tear drops
- Emotional title and content reveal
- Reset / try again button
- Responsive design
